# 🛡️ 支援士クエスト

情報処理安全確保支援士（SC）試験の午前II対策を、ゲーム感覚で毎日続けられる学習アプリです。

**▶ https://tarepon.github.io/sc-quest/** （スマホ推奨・「ホーム画面に追加」でアプリのように使えます）

## 特徴

- オリジナル問題 全123問・7分野（暗号技術／認証・認可／攻撃手法／ネットワーク防御／セキュア開発／管理・法規／運用・インシデント対応）
- 間隔反復・検索練習など学習科学に基づく出題エンジン（1→3→7→14→30日の復習スケジュール）
- XP・レベル・連続日数・バッジのゲーミフィケーション
- 本番形式の午前II模擬試験モード（25問・60%合格判定）
- 同期コードによる端末間の実績引き継ぎ（iPhone⇄iPad）

## プライバシー

学習記録は利用端末のブラウザ内（localStorage）にのみ保存されます。外部サーバーへのデータ送信・アクセス解析は一切ありません。

## 免責・商標

- 本アプリは個人が作成した**非公式**の学習ツールであり、独立行政法人情報処理推進機構（IPA）および試験実施機関とは一切関係ありません。
- 収録している問題・解説はすべてオリジナルであり、試験問題の転載ではありません。
- 内容の正確性・最新性は保証されません。受験手続・制度の確認は必ず [IPA公式サイト](https://www.ipa.go.jp/) で行ってください。
- 「情報処理安全確保支援士」は国家資格の名称です。「登録セキスペ」は独立行政法人情報処理推進機構（IPA）の登録商標です。
